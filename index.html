OBJECT DETECTION AND OBJECT TRACKING WITH VOICE FEEDBACK

Aim

Object detection and tracking with voice feedback is a challenging task in the field of computer vision.
The goal is to develop a system that can detect and track objects in realtime. The system should also provide voice feedback to the user in real-time.



Proposed System

* Object detection is a computer vision task that involves identifying and locating objects in images or videos.
*It is an important part of many applications, such as surveillance, self-driving cars, or robotics.
*YOLO (You Only Look Once) is an object detection algorithm that identifies and localize objects in an image or video by dividing it
into a grid and predicting bounding boxes and class probabilities for each grid cell, in single forward pass through a neural network.
*Voice Output: Speech synthesis is the artificial production of human speech.
A text-to-speech (specifically pyttsx3 library in python) is used which converts normal language text into speech.
